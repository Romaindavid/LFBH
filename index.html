Rigueur et Transparence

Source des données : API officielle Flightradar24 (fr24api.flightradar24.com), endpoint flight-summary, filtré sur l'aéroport LFBH.

Fréquence de mise à jour : Le plan API souscrit limite l'historique disponible à 29 jours glissants. Régénérer les données nécessite de relancer la collecte manuellement.

Limites techniques : Certains appareils (notamment militaires) peuvent apparaître sans immatriculation ni opérateur communiqué. L'estimation CO2 est un ordre de grandeur, pas une mesure : elle ne tient pas compte du taux de remplissage, des conditions météo, ni des trajectoires réelles.

Code source : Pipeline de collecte et scripts d'analyse disponibles sur github.com/Romaindavid/LFBH.
